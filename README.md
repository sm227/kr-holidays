# 대한민국 법정 기념일 캘린더

각종 기념일 등에 관한 규정(57개)과 개별법(107개)에 따른 기념일 전체 164개를 담은 iCal 구독 파일입니다.

## 구독 방법

아래 링크를 클릭하면 캘린더 앱에서 바로 구독할 수 있습니다.

<a href="webcal://raw.githubusercontent.com/hamsm3184/kr-holidays/main/korean_memorial_days_all.ics">캘린더 구독하기</a>

> 링크가 열리지 않으면 캘린더 앱에서 직접 URL을 입력하세요.  
> `https://raw.githubusercontent.com/hamsm3184/kr-holidays/main/korean_memorial_days_all.ics`

## 포함 기념일

- 각종 기념일 등에 관한 규정 기념일 57개
- 개별법에 규정된 기념일 107개

## 참고

- 음력 기념일(씨름의 날, 이산가족의 날)은 2026년 기준 양력 날짜로 등록되어 있어 매년 수동 조정이 필요합니다.
- 요일 기준 기념일(성년의 날, 문화의 날 등)은 매년 자동으로 정확한 날짜에 맞춰집니다.
- 해양안전의 날, 안전점검의 날은 매월 반복됩니다.
